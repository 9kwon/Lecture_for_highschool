# GitHub 업로드 & 강의 링크 만들기 (Private + GitHub Pages)

게시용 폴더는 이미 준비됐습니다: `ai-language-lecture/`
(`index.html`, `assets/`(25개), `README.md`, `.gitignore`)

아래 순서대로 진행하면 **링크 클릭만으로 강의 페이지가 열립니다.**

---

## 1. GitHub에서 빈 저장소 만들기

1. https://github.com/new 접속
2. **Repository name**: `ai-language-lecture`
3. **Private** 선택
4. README/.gitignore **추가하지 않음** (이미 폴더에 있음)
5. **Create repository** 클릭

> 생성 후 나오는 주소(`https://github.com/<사용자명>/ai-language-lecture.git`)를 복사해 두세요.

---

## 2. 내 PC에서 업로드 (PowerShell 또는 Git Bash)

폴더로 이동한 뒤, `<사용자명>` 부분만 본인 것으로 바꿔서 실행하세요.

```bash
cd "C:\Users\MMM\Desktop\High_school\ai-language-lecture"

git init
git add -A
git commit -m "특강: AI를 부리는 언어 강의 자료 게시"
git branch -M main
git remote add origin https://github.com/<사용자명>/ai-language-lecture.git
git push -u origin main
```

> 처음 push 시 GitHub 로그인 창이 뜨면 계정으로 인증하면 됩니다.

---

## 3. GitHub Pages 켜기 (링크 생성)

1. 저장소 → **Settings** → 왼쪽 **Pages**
2. **Source**: `Deploy from a branch`
3. **Branch**: `main` / 폴더 `/ (root)` → **Save**
4. 1~2분 뒤 다음 주소에서 강의 페이지가 열립니다:

```
https://<사용자명>.github.io/ai-language-lecture/
```

이 링크를 외부 수강생에게 공유하면 됩니다.

---

## ⚠️ 중요: Private 저장소 + Pages

GitHub **무료(Free) 계정은 Private 저장소에서 Pages가 비활성화**됩니다.
Private 상태로 강의 링크를 공개하려면 둘 중 하나가 필요합니다:

- **GitHub Pro 이상** 유료 플랜 → Private 저장소에서도 Pages 사용 가능
- 또는 저장소를 **Public으로 전환** (Settings 맨 아래 *Change visibility*)
  → 무료로 Pages 링크 생성 가능

> 자료 자체를 외부 수강생에게 공유할 목적이라면, Public 전환이 가장 간단하고 무료입니다.
> 코드/링크를 아는 사람만 보면 되므로 Public이어도 검색 노출 위험은 낮습니다.
