# 특강 · AI를 부리는 언어

외부 강의용 특강 자료입니다. `index.html`이 강의 페이지 본문이며, 이미지·영상은 `assets/` 폴더에 있습니다.

## 바로 보기

GitHub Pages가 활성화되면 아래 주소에서 링크 클릭만으로 강의 페이지가 열립니다:

```
https://<your-username>.github.io/ai-language-lecture/
```

## 로컬에서 보기

저장소를 내려받은 뒤 `index.html`을 브라우저로 열면 됩니다.

## 구성

- `index.html` — 강의 본문
- `assets/` — 이미지, GIF, 동영상 등 강의에 사용된 자료
